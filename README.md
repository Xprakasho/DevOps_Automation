
🚀 DevOps Automation Bootcamp

A structured engineering portfolio documenting the journey from Telecom Cloud Engineer to Cloud Platform & DevOps Engineer through hands-on projects, automation, cloud infrastructure, Kubernetes, GitOps, AWS, and AI integration.

🎯 Project Vision

This repository is designed to help engineers understand modern DevOps and Cloud technologies by combining theory with practical implementation.

Instead of learning isolated tools, every module contributes to building a complete production-style platform.

The goal is not only to learn technologies, but also to understand:

Why they exist
What problems they solve
How they work internally
Where they fit in real production environments

📖 Learning Philosophy

Every technology in this repository follows the same engineering process:

Understand
      ↓
Build
      ↓
Break
      ↓
Debug
      ↓
Document
      ↓
Automate

This approach develops practical engineering skills instead of memorizing commands.
